제품 개요

XX-MSR는 마그네틱 스트라이프 리더(MSR)와 스마트 카드 리더(SCR) 기능을 단일 장치에 통합한 다기능 장치로, 마그네틱 카드 읽기 및 IC 카드 읽기/쓰기 기능을 지원합니다.

하드웨어 구조

XX-MSR는 Alcor Micro의 USB 허브 칩(VID: 0x058F)을 활용하여 구성 요소와 호스트 PC 간의 연결을 지원합니다. 하드웨어는 두 가지 주요 부분으로 나뉩니다:

  • MSR 파트: 마그네틱 카드 읽기 기능은 마이크로컨트롤러(MCU)가 UART를 통해 FTDI의 USB-시리얼 어댑터 칩에 연결되고, 이 어댑터는 USB 허브 칩을 통해 PC에 연결됩니다.
  • SCR 파트: IC 카드 읽기/쓰기 기능은 Alcor Micro의 PC/SC 카드 리더 모듈이 USB 허브 칩에 직접 연결되어 PC와 통신합니다.

소프트웨어 지원 구조

XX-MSR는 두 가지 기능에 맞춘 소프트웨어 지원을 제공합니다:

  • MSR 파트: 이 장치는 mrwjslib.dll라는 32비트 동적 링크 라이브러리(DLL)를 제공하며, 이 DLL은 마그네틱 카드 작업을 위한 다음 함수들을 내보냅니다:
    • MSRW_CDMSRead
    • MSRW_CardDataRead
    • MSRW_CheckStatus
    • MSRW_Close
    • MSRW_Open
    • MSRW_PBMSRead
    • MSRW_PBMSWrite
    • MSRW_PassbookDataRead
    • MSRW_Reset
  • SCR 파트: IC 카드 기능은 Windows CCID 드라이버를 활용하며, PC/SC 호환 스마트 카드 작업을 위해 표준 Winscard 라이브러리를 사용하는 것으로 추정됩니다.

검토 결과

하드웨어

단종된 lpu237 E type rev G의 내부 USB 허브에 핀패드에 사용 중인 EMV IC 모듈을 USB 연결하면 대체 가능. 단 KC 인증 필요.

펌웨어

가상 COM을 지원하기 위해 lpu238 (europa v1.3)용 펌웨어를 사용하고, 내부적으로 사용되는 프로토콜을 얻어서 수정해야 함. (마이컴 메모리 공간 부족으로 수정 후 가능성 최종 판단 가능)

소프트웨어

  • SCR 파트: T0 working time extension 처리 부분만 같다면, 기존 프로그램을 사용 가능.
  • MSR 파트: 현재 사용 중인 mrwjslib.dll의 소스 또는 함수 프로토타입 및 기능 설명서를 얻어 동일 기능을 하는 DLL을 제작해야 함.
